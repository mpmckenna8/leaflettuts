Welcome to some of my attempts at using Leaflet.

Below are some attempts at making stuff based on the leaflet.js examples:
  • See my version of the Quick Start Leaflet tutorial but fanagled so you don't need an API Key and use Stamen Base Map: Matt's start on the leaflet Quick Start Tutorial.
  • Solo maptime effort by Meghan Hade practicing w/ leaflet.js to make a Dragends map.
  • Because I had qualms with the marker starting in that state with fewer lakes than Wisconsin I decided to try out a Stamen basemap with a custom marker in Undraggable Sconnie Dragon map.
